# 玫瑰
核心寓意：爱情、浪漫、忠贞，是表达深情的经典选择
适配场景：情人节、求婚、纪念日、表白
常见品种：红玫瑰（热恋）、粉玫瑰（温柔爱意）、白玫瑰（纯洁告白）、香槟玫瑰（优雅守护）
养护方法：
1. 收到花后斜剪花茎45°，增加吸水面积
2. 去除花茎下部叶片（避免浸泡腐烂）
3. 每日更换清水，加入鲜花保鲜剂效果更佳
4. 放置于阴凉通风处，远离阳光直射和空调出风口
5. 保鲜期：5-7天
价格区间：
- 单束（11朵）：红玫瑰50元、粉玫瑰45元、白玫瑰48元
- 礼盒装（33朵）：红玫瑰159元、香槟玫瑰179元
- 高端定制（99朵）：399-599元（含包装和配送）

# 康乃馨
核心寓意：感恩、母爱、尊敬、祝福，象征温柔与关怀
适配场景：母亲节、教师节、探望长辈、表达谢意
常见品种：粉康乃馨（感恩）、白康乃馨（纯洁祝福）、红康乃馨（热烈关爱）
养护方法：
1. 花茎斜剪30°，长度根据花瓶高度调整（露出水面10-15cm）
2. 水温控制在20℃左右，避免冰水或热水
3. 去除下部叶片，仅保留花头下方2-3片叶子
4. 每2天更换一次清水，清洗花瓶内壁
5. 保鲜期：7-10天
价格区间：
- 单束（19朵）：粉康乃馨30元、白康乃馨28元
- 礼盒装（33朵）：89-129元（含丝带和贺卡）
- 组合花束（康乃馨+满天星）：45-69元

# 百合
核心寓意：纯洁、高雅、百年好合，象征圆满与祝福
适配场景：婚礼、乔迁、开业、探望病人
常见品种：白百合（纯洁）、粉百合（浪漫）、黄百合（友谊）、香水百合（浓香）
养护方法：
1. 深水养护，花茎浸入水中10cm，每天修剪1cm花茎
2. 去除花头下方的叶片，避免接触水面
3. 保持水质清洁，每1-2天换水一次
4. 香水百合香味浓郁，避免放置于密闭空间
5. 保鲜期：10-15天（香水百合稍短，8-12天）
价格区间：
- 单束（5-7朵）：白百合60元、粉百合65元、香水百合75元
- 婚礼花束（百合+玫瑰）：199-299元
- 盆栽百合（带花苞）：89-159元

# 向日葵
核心寓意：阳光、希望、勇敢、积极向上，象征活力与鼓励
适配场景：生日、毕业季、职场祝福、探望朋友
常见品种：观赏向日葵、迷你向日葵、重瓣向日葵
养护方法：
1. 花茎粗壮，斜剪45°，去除下部所有叶片
2. 花瓶注入3/4清水，加入保鲜剂延长花期
3. 放置于有散射光的地方，避免强光直射
4. 每日检查水质，浑浊时及时更换
5. 保鲜期：5-8天
价格区间：
- 单束（3-5朵）：45元
- 组合花束（向日葵+小雏菊）：69-99元
- 礼盒装（9朵）：129-159元

# 郁金香

- 进口郁金香（稀有品种）：120-180元/束

# 小雏菊
核心寓意：快乐、纯洁、深藏的爱，象征天真与美好
适配场景：生日、日常赠礼、搭配其他花束、装饰家居
常见品种：白色小雏菊、粉色小雏菊、黄色小雏菊、混色小雏菊
养护方法：
1. 花茎较细，斜剪45°，长度根据花瓶调整（露出水面8-10cm）
2. 去除下部叶片，避免叶片接触水面
3. 清水没过花茎底部即可，无需深水
4. 每2天更换一次清水，修剪花茎末端
5. 保鲜期：8-12天
价格区间：
- 单束（10-15朵）：25-35元
- 组合花束（小雏菊+满天星）：39-59元
- 花瓶套装（花束+简约花瓶）：69-99元

# 满天星
核心寓意：思念、陪伴、默默无闻的爱，常作为配花
适配场景：搭配主花束、装饰礼盒、单独赠送（小束）
常见品种：白色满天星、粉色满天星、蓝色满天星（染色）、紫色满天星
养护方法：
1. 花茎细且多，斜剪45°，去除下部多余枝叶
2. 花瓶注入1/2清水，花茎浸入5-8cm
3. 放置于阴凉通风处，避免阳光直射
4. 每3天更换一次清水，清洗花瓶
5. 保鲜期：10-15天（干花可长期保存）
价格区间：
- 小束满天星（配花）：15-25元
- 单束满天星（100g）：39-59元
- 干花满天星（礼盒装）：69-99元

# 绣球花
核心寓意：团圆、美满、希望，象征幸福与和谐
适配场景：婚礼、乔迁、开业、家庭装饰
常见品种：蓝色绣球、粉色绣球、白色绣球、紫色绣球
养护方法：
1. 花茎较粗，斜剪45°，底部十字剪口（增加吸水）
2. 深水养护，花茎浸入水中10-15cm
3. 去除下部叶片，避免腐烂污染水质
4. 放置于阴凉处，避免阳光直射和高温
5. 每天更换清水，修剪花茎末端
6. 保鲜期：7-10天
价格区间：
- 单支绣球（大花头）：35-50元
- 花束（3-5支）：129-199元
- 盆栽绣球（带花）：159-299元
